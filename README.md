# Data Analytics Projects
# Project Objective

This project explores customer flight behaviour and loyalty patterns using exploratory data analysis techniques. The objective was to identify trends, behavioural differences, and potential customer segmentation insights based on flight activity, distance travelled, and loyalty program engagement.

Tools Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
Key Tasks
Data cleaning and preprocessing
Missing value and duplicate analysis
Data type conversion
Exploratory Data Analysis (EDA)
Distribution and outlier analysis
Correlation analysis
Time-series trend visualization
Comparative analysis between customer groups
Dataset Overview

The dataset contains customer flight activity and loyalty-related information between 2019 and 2021, including:

Number of flights
Distance travelled
Loyalty points accumulated and redeemed
Companion flights
Monthly activity trends
Enrollment type information

The analysis was performed on datasets containing more than 600,000 observations.

Key Insights
Most customers redeem very few loyalty points
Flight activity and travelled distance show strong positive correlation
Seasonal patterns are visible across flight activity
Customer groups display different flight engagement behaviours
